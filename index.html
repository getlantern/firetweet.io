#UnblockedTweeting

An Android app powered by Lantern that gives direct unblocked access to Twitter from anywhere in the world.

FireTweet is an Android app powered by Lantern that gives direct unblocked access to Twitter from anywhere in the world.

Features include:

Easy-to-use: works without requiring the device be configured in any additional way.

Gives the full experience of Twitter to those in regions where Twitter is blocked.

Integrates Lantern as a shared library, and using a small segment of code updates the application to use a proxy server on start.

All HTTP traffic is routed through a distributed network of direct proxies.

No ads!

All Twitter API requests are processed through Lantern.
